# Indonesian Working Day
This library functions to provide an effective Indonesian business day calendar.

## For Install
```bash
pip3 install git+https://github.com/irsyad.damlis/id_workingday.git
```

## For Update
```bash
pip3 uninstall -y id_workingday && pip3 install --no-cache-dir git+https://github.com/irsyaddamlis/id_workingday.git
```
